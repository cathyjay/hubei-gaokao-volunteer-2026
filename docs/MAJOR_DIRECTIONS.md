# 专业方向研究

最后更新：2026-06-25

## 一、当前已提出方向

目前家庭先想到的方向如下：

| 方向 | 当前状态 | 后续必须回答的问题 |
| --- | --- | --- |
| 数字媒体技术 | 初始关注 | 是偏技术、偏设计还是偏内容？是否接受编程、数学、作品集和就业竞争？ |
| 师范类专业 | 初始关注 | 是想当中小学老师、学前/小学教育、英语/数学/物理/化学/地理师范，还是教育技术？是否接受考编压力？ |
| 计算机类相关专业 | 初始关注 | 是否能接受较强数学、编程和持续学习？是否能接受民办/中外合作/较弱学校换专业？ |

这些方向都还不是最终结论。下一步要继续补充“可以接受、勉强接受、不能接受”的专业清单。

## 二、专业代码初步核验

根据教育部 2026 本科专业目录和千问高考公开专业分类接口，当前相关专业可先按以下名称追踪，后续以教育部目录和 2026 湖北招生计划为准。

| 专业或专业类 | 代码 | 备注 |
| --- | --- | --- |
| 计算机类 | 0809 | 专业类，不是具体专业 |
| 计算机科学与技术 | 080901 | 计算机核心专业，数学和编程要求较高 |
| 软件工程 | 080902 | 关注学费、校企合作和实践要求 |
| 数字媒体技术 | 080906 | 通常在计算机类下，偏技术，不等于数字媒体艺术 |
| 物联网工程 | 080905 | 可能涉及硬件、通信、嵌入式 |
| 数据科学与大数据技术 | 080910T | 数学、统计和编程要求较高 |
| 网络工程 | 080903 | 关注学校平台和实践环境 |
| 教育学类 | 0401 | 专业类，不等于所有“师范”专业 |
| 小学教育 | 040107 | 常见师范方向，需看就业和考编地区 |
| 学前教育 | 040106 | 就业路径明确，但需确认是否接受工作内容 |
| 教育技术学 | 040104 | 可能与技术、教育信息化交叉 |

注意：师范类不是一个单一专业。很多“某某师范大学”的非师范专业不一定发教师教育培养；很多综合大学也有师范专业。最终要看招生计划备注、培养方案和学校招生章程。

## 三、初步判断提示

- 数字媒体技术和数字媒体艺术不同。前者通常偏计算机和技术，后者通常偏艺术设计，招生类别和培养路径不同。
- 计算机类热度高，分数和调剂风险都要谨慎。该位次如果追求计算机，可能要在学校层次、城市、公办属性之间做取舍。
- 师范类要看具体学科。英语 122 是优势，可以研究英语师范、教育技术、小学教育等；数学 88、物理 50 提示数学/物理师范要谨慎。
- 地理 81 可以关注地理科学、地理信息科学、城乡规划、自然地理与资源环境等相关方向，但要核验选科和就业路径。
- 不要只按“专业名字好听”选择，要看课程、就业和调剂后的最差情况。

## 四、下一轮需要家庭讨论

- 考生本人对编程、设计、做老师、考编、考研的真实接受度。
- 是否接受民办、独立学院、职业本科、中外合作。
- 是否接受护理、康复、药学、食品、化工、材料、土木、环境、农学、生物类。
- 是否接受非偏好城市换更稳的公办和更好的专业。
- 对“学校优先、城市优先、专业优先、公办优先”的排序。

## 五、专业研究记录规则

每新增一个专业方向，都要记录：

- 专业名称和代码。
- 所属专业类。
- 主要课程和学习难度。
- 就业出口和升学必要性。
- 对数学、物理、英语、编程、表达或动手能力的要求。
- 湖北 2026 有哪些院校专业组招生。
- 近三年投档位次。
- 能否接受被调剂到同组其他专业。
