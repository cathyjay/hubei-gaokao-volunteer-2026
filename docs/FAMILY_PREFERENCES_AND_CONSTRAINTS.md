# 家庭偏好与底线

最后更新：2026-06-26

## 一、当前结论

本文件记录当前家庭偏好和硬底线，用于筛选院校专业组。后续如果家庭意见变化，需要在 `docs/DECISIONS.md` 记录变更原因和影响范围。

| 维度 | 当前口径 | 类型 | 筛选含义 |
| --- | --- | --- | --- |
| 学校属性 | 优先公办本科 | 硬约束偏强 | 公办普通本科进入主方案；民办、独立学院、职业本科暂不作为常规选择 |
| 学费预算 | 公办、正常学费，年学费上限暂按 15000 元 | 硬约束偏强 | 普通公办学费优先；年学费超过 15000 元默认不进入主方案 |
| 中外合作 | 暂不作为常规选择 | 硬约束偏强 | 除非后续家庭明确接受，否则不进入主方案 |
| 民办/独立学院 | 暂不作为常规选择 | 硬约束偏强 | 只有在保本科压力不足或有明显优势时再单独讨论 |
| 专业理解 | 目前不了解专业 | 待研究 | 不以专业名称好听作为依据，必须研究课程、就业、升学和调剂风险 |
| 明确不接受 | 医学类、护理类 | 硬约束 | 主方案不纳入医学、护理方向；组内如含医学/护理且可能调剂，需谨慎或排除 |
| 当前专业优先级 | 数字媒体技术 > 计算机类 > 师范类 | 软偏好 | 先按此顺序寻找可接受专业组；其他方向继续了解，不提前排除 |
| 是否愿意当老师 | 可以了解 | 待研究 | 师范类可以进入研究范围，但要继续看学科、考编压力、就业地区和是否真愿意从教 |
| 排序倾向 | 专业和学校优先，城市其次 | 决策原则 | 先保证专业组可接受、学校属性和费用可接受，再按城市排序 |
| 城市偏好 | 成都、西安、武汉、北京 | 软偏好 | 按此顺序作为初始城市权重，后续可扩展其他城市 |
| 色盲/色弱 | 无色盲 | 体检信息 | 色觉限制风险较低，但仍以高考体检结果和章程为准 |
| 近视 | 有近视，度数待补充 | 体检待补充 | 需确认是否高度近视；公安、军校、航海、飞行、部分医学等需谨慎 |

## 二、当前默认过滤规则

第一轮筛选时，优先保留：

- 公办本科。
- 普通学费，且年学费不超过 15000 元。
- 专业组内没有明确不能接受的专业。
- 优先专业方向依次为：数字媒体技术、计算机类相关专业、师范类专业。
- 选科匹配物理、化学、地理。
- 近三年等位分和位次处于可冲、可稳、可保范围。
- 城市优先级靠前，或学校/专业明显更适合。

第一轮暂缓或排除：

- 医学类、护理类专业组；如果一个专业组内存在医学/护理且服从调剂可能进入，也要谨慎或排除。
- 明显高收费专业组。
- 年学费超过 15000 元的专业组。
- 中外合作专业组。
- 民办、独立学院、职业本科。
- 组内存在明显不能接受专业且必须服从调剂的专业组。
- 需要特殊资格、定向服务、专项资格而当前不确定的专业组。
- 与近视、体检、单科、语种要求冲突的专业组。

## 三、需要继续补充的信息

- 近视度数和高考体检结论。
- 是否完全排除民办、独立学院、职业本科，还是仅作为最后保底备选。
- 年学费上限：暂按 15000 元处理；超过该上限默认不进主方案。
- 不能接受专业清单：当前已明确医学、护理不接受，其他专业待了解。
- 对数字媒体技术、计算机类、师范类的真实接受度和替代方向。
- 是否接受省内非武汉城市，或重庆、长沙、合肥、南昌、郑州、南京周边等城市。

## 四、决策优先级

当前建议采用以下优先级：

```text
1. 不滑档、不退档
2. 专业组内所有专业可接受
3. 公办和普通学费
4. 学校和专业质量
5. 城市偏好
```

城市会影响排序，但不能让我们接受不能读、不能调剂或费用明显超预算的院校专业组。
