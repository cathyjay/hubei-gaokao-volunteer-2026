# 家庭偏好与底线

最后更新：2026-06-28

本文件记录当前家庭偏好、费用边界、专业底线和体检使用原则。后续如果家庭意见变化，需要在 `docs/DECISIONS.md` 记录变更原因和影响范围。

## 一、当前结论

| 维度 | 当前口径 | 类型 | 筛选含义 |
| --- | --- | --- | --- |
| 学校属性 | 优先公办本科 | 硬约束偏强 | 公办普通本科仍是主线；民办、独立学院、职业本科暂不作为常规选择 |
| 普通主线学费 | 公办、正常学费优先 | 硬约束偏强 | 普通公办低费用方案仍要保留，用于稳妥主线 |
| 扩展费用上限 | 中外合作或高收费项目年学费最高可接受 70000 元 | 专项讨论 | 可用更高学费换取更好学校或更合适专业机会，但必须单独核验培养模式、证书、校区、出国要求和调剂风险 |
| 中外合作 | 可进入专项讨论 | 软硬结合 | 不再默认排除；超过 70000 元默认不进入家庭讨论 |
| 民办/独立学院 | 暂不作为常规选择 | 硬约束偏强 | 只有保本科压力不足或确有明显优势时再单列讨论 |
| 专业理解 | 目前仍在了解专业 | 待研究 | 不以专业名称好听作为依据，必须研究课程、就业、升学和调剂风险 |
| 明确不接受 | 医学类、护理类 | 硬约束 | 主方案不纳入医学、护理方向；医学影像、医学检验来自外部建议，暂列待确认例外 |
| 当前专业优先级 | 数字媒体技术 > 计算机类 > 软件/人工智能/网络/电子信息/自动化/机器人工程 > 师范类 > 环境/农业/工商管理等待了解方向 | 软偏好 | 先按此顺序寻找可接受专业组；其他方向继续了解，不提前排除 |
| 是否愿意当老师 | 可以了解 | 待研究 | 师范类可以进入研究范围，但要继续看学科、考编压力、就业地区和是否真愿意从教 |
| 排序倾向 | 专业和学校优先，城市其次 | 决策原则 | 先保证专业组可接受、学校属性和费用可接受，再按城市排序 |
| 城市偏好 | 成都、西安、武汉、北京 | 软偏好 | 按此顺序作为初始城市权重，后续可扩展其他城市 |
| 色盲/色弱 | 体检截图显示色觉相关检查正常 | 体检信息 | 公开仓库只记录摘要；具体体检明细保存在 git 忽略的本地私有目录 |
| 近视 | 有近视 | 体检约束 | 军校、公安、航海、飞行、部分医学和其他限报专业必须按官方体检标准逐项核验 |
| 军事院校 | 只作低优先级政策核验 | 特殊批次 | 2026 湖北首选物理特殊招生线为 529，当前 515 低于该线；军校通知要求达到特殊招生控制线后再按规则确定参检，因此暂不作为主线 |

## 二、当前两条筛选口径

### 1. 普通低费用主线

第一轮普通主线仍优先保留：

- 公办本科。
- 普通学费，优先低费用。
- 专业组内没有明确不能接受的专业。
- 优先专业方向依次为：数字媒体技术、计算机类相关专业、软件工程/人工智能/网络工程/电子信息/自动化/机器人工程、师范类专业。
- 选科匹配物理、化学、地理。
- 近三年等位分和位次处于可冲、可稳、可保范围。
- 城市优先级靠前，或学校/专业明显更适合。

普通主线暂缓或排除：

- 医学类、护理类专业组；如果一个专业组内存在医学/护理且服从调剂可能进入，也要谨慎或排除。
- 年学费明显高于普通公办且没有学校/专业优势的专业组。
- 民办、独立学院、职业本科。
- 组内存在明显不能接受专业且必须服从调剂的专业组。
- 需要特殊资格、定向服务、专项资格而当前不确定的专业组。
- 与近视、体检、单科、语种要求冲突的专业组。

### 2. 7 万预算中外合作专项线

新增专项口径记录在：

- `data/working/family-preferences-expanded-2026-06-28.json`
- `data/exports/issue19-expanded-budget-coop-scenario.xlsx`
- `data/exports/issue19-expanded-budget-coop-groups.csv`
- `data/exports/issue19-expanded-budget-coop-majors.csv`

专项线只用于“是否值得花钱买更好学校或更优机会”的讨论，不自动进入最终方案。进入专项线的项目必须逐项核：

- 是否确为 2026 湖北物理类招生计划。
- 是否在同一个院校专业组内。
- 年学费、住宿费、出国阶段费用和汇率风险。
- 培养模式：4+0、3+1、2+2、双学位、单证或中外双证。
- 是否限制转专业、是否只录取有明确专业志愿考生。
- 校区、语言要求、外语单科要求、英文授课比例。
- 毕业证、学位证、合作高校认可度。
- 组内所有专业是否都能接受调剂。

## 三、体检和特殊批处理原则

体检截图含敏感身份和健康信息，公开仓库只保留决策摘要，不保存姓名、考生号、体检截图或完整明细。

当前公开使用结论：

- 色觉相关检查正常。
- 有近视。
- 身高、体重和其他体检明细只作为私有核验材料。

军事院校、公安院校、航海、飞行、定向培养军士、部分医学及其他限报专业，不能只看普通高考体检截图。若进入讨论，必须按当年官方通知和招生章程逐条核验。

当前军校初判：考生 515 分低于 2026 湖北首选物理特殊招生线 529 分；湖北 2026 军队院校招生通知写明，参检人员从达到普通类本科特殊招生录取控制分数线且政审通过等条件的考生中确定。因此军校暂不作为可执行主线，只保留政策核验和特殊变化观察。

## 四、需要继续补充的信息

- 是否把医学影像、医学检验完全排除，还是允许作为非临床医学的例外方向了解。
- 是否完全排除民办、独立学院、职业本科，还是仅作为最后保底备选。
- 对数字媒体技术、计算机类、师范类、自动化/机器人/电子信息、环境/农业/工商管理的真实接受度和替代方向。
- 是否接受省内非武汉城市，或重庆、长沙、合肥、南昌、郑州、南京周边等城市。
- 对中外合作项目的证书、出国、英文授课和不能转专业风险的接受程度。

## 五、决策优先级

当前建议采用以下优先级：

```text
1. 不滑档、不退档
2. 专业组内所有可能被录取或调剂的专业可接受
3. 费用和培养模式可承受、可理解
4. 学校和专业质量
5. 城市偏好
```

城市会影响排序，但不能让我们接受不能读、不能调剂、费用不可承受或特殊要求不满足的院校专业组。
