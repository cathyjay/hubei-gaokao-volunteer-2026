## 二、湖北本科普通批结构

本科普通批以“院校专业组”为基本填报和投档单位。

- 最多可填报 45 个院校专业组志愿。
- 每个院校专业组内最多可选报 6 个专业。
- 每个院校专业组内还要选择是否服从专业调剂。
- 平行志愿投档遵循“分数优先、遵循志愿、一次投档、不再补档”。

这意味着：一旦投档到某个院校专业组，本批次后续院校专业组志愿不再检索。如果进档后被退档，不能再投同批次后面的其他院校专业组，只能等待征集志愿或后续批次。

## 三、必须理解的专业名词

| 名词 | 含义 | 本项目里的处理方式 |
| --- | --- | --- |
| 院校专业组 | 同一高校按选科和专业组合拆出的志愿单位 | 每个组单独判断，不用学校整体分替代 |
| 专业组代码 | 湖北招生计划中院校专业组的代码 | 最终表必须记录 2026 代码 |
| 再选科目要求 | 专业组或专业要求的化学、生物、地理等条件 | 考生为化学、地理；没有生物，需逐项核验 |
| 一分一段 | 每个分数对应同分人数和累计人数 | 固定位次和等位分的基础 |
| 位次 | 考生在全省同类考生中的排序位置 | 本项目使用累计位次 91723 和区间 90895-91723 |
| 等位分 | 按相近位次把 2026 分数映射到往年分数 | 2025 约 494，2024 约 497，2023 约 481 |
| 投档线 | 某院校专业组当年投档最低分或最低位次 | 只能说明进组门槛，不代表能进组内任意专业 |
| 省控线 | 批次最低控制分数线 | 2026 首选物理本科线 435 |
| 特招线 | 特殊类型招生控制线 | 2026 首选物理特招线 529 |
| 招生计划 | 当年在湖北招多少人、招哪些专业 | 最终判断必须看 2026 计划 |
| 招生章程 | 高校录取规则、限制条件、收费等正式文件 | 每个最终候选必须核章程 |
| 平行志愿 | 同批次多个志愿按分数和顺序检索 | 顺序仍重要，不能乱放不想去的组 |
| 服从调剂 | 组内所填专业未录取时，同意调到组内其他有计划专业 | 只在能接受组内所有可能调剂专业时选择 |
| 滑档 | 分数/位次不够，未被任何志愿投档 | 需要足够稳和保的志愿 |
| 退档 | 已投档但因不符合条件或专业无法录取被退回 | 需要核体检、单科、语种、调剂和章程 |
| 征集志愿 | 某批次录取后未完成计划再次征集 | 只能作为兜底，不作为正常策略 |
| 冲稳保 | 按风险层次排列志愿 | 必须结合位次、专业组、调剂和计划人数判断 |

## 四、填报注意事项

1. 不用 2025 裸分 515 直接找学校。跨年比较必须以位次和等位分为主。
2. 院校专业组不是学校整体。一个学校不同专业组的分数、专业质量和风险可能差别很大。
3. 服从调剂前要逐个看组内专业。只要有一个绝对不能接受的专业，就不能轻易选择服从调剂。
4. 不服从调剂会增加退档风险。只有在专业录取把握较高、章程风险可控时才考虑。
5. 专业组代码和专业组成每年可能变化。2023-2025 同名学校不能机械合并到 2026。
6. 计划人数少的专业组波动大，不能作为唯一保底。
7. 中外合作、民办、独立学院、职业本科、预科、民族班、专项计划要分开判断，不能只看学校名。
8. 必须核验学费、校区、培养模式、联合培养、是否出国、是否有服务年限。
9. 必须核验体检限报、色盲色弱、视力、身高、语种和单科成绩要求。
10. 北京等高热城市要冷静。该位次附近可能伴随高费用、民办、冷门专业或特殊备注。
11. 第三方工具的推荐概率只能做参考，不能替代官方投档线、2026 招生计划和招生章程。
12. 最终提交后要再次登录查询志愿是否保存成功，并保留提交核对记录。

## 五、最终填报前检查清单

- 45 个院校专业组是否按“最想去且风险可接受”到“可靠保底”排序。
- 每个院校专业组是否有 2023、2024、2025 数据或明确缺失原因。
- 每个院校专业组是否核过 2026 湖北招生计划。
- 每个院校专业组是否核过招生章程。
- 每个院校专业组内的 6 个专业排序是否明确。
- 是否记录了可接受、勉强接受、不能接受专业。
- 是否逐个确认服从调剂结论。
- 是否有足够保底，且保底不是小计划、高风险、不能接受专业组。
- 是否核对考生选科：物理、化学、地理。
- 是否核对系统保存、提交、退出、再次登录查询。

## 六、变更规则

2026-06-30 后不再随意新增方向。任何修改必须记录：

- 修改时间。
- 修改原因。
- 信息来源。
- 影响的院校专业组。
- 对冲稳保结构和保底强度的影响。

2026-07-02 12:00 后只处理应急事项，例如官方系统显示与方案不一致、招生计划核验发现重大错误、登录保存异常等。
