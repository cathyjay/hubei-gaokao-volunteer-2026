## 二、目标

我们要在 2026-06-30 前完成所有关键决策和最终志愿方案，2026-07-02 12:00 前只保留应急修改缓冲。官方本科普通批集中填报截止时间仍为 2026-07-02 17:00，项目内部提前收口是为了给登录、核对、家庭确认和突发情况留出安全余量。

核心目标：

1. 保证录取成功率，避免滑档、退档或被明显不能接受的专业锁定。
2. 尽量用足 515 分、91723 位次，不做过度保守选择。
3. 在学校、城市、专业、就业、升学、费用、校区和调剂风险之间做清楚权衡。
4. 所有结论可复查，有原始来源、派生数据、人工复核记录和决策理由。

## 三、已知偏好

当前城市偏好顺序：

- 成都
- 西安
- 武汉
- 北京

这些只是初始城市，不是最终边界。后续可以增加南京、重庆、长沙、合肥、郑州、南昌、杭州周边等城市；每次增加城市，都应重新跑筛选脚本并记录筛选结果。

当前家庭偏好与底线：

- 学校属性：优先公办本科。
- 预算：正常学费预算，普通公办学费优先。
- 民办、独立学院、职业本科、中外合作、高收费专业组暂不作为常规选择。
- 排序倾向：专业和学校优先，城市其次。
- 体检信息：无色盲；有近视，度数和高考体检结论待补充。
- 专业理解：目前不了解专业，需要继续研究。

详细记录见 `docs/FAMILY_PREFERENCES_AND_CONSTRAINTS.md` 和 `data/working/family-preferences.json`。

当前提到的可能专业方向：

- 数字媒体技术
- 师范类专业
- 计算机类相关专业

这些只是起点，不是最终结论。因为目前对专业和学校都不了解，后续需要从课程难度、就业出口、升学必要性、考编考证、城市资源、学费、调剂风险等角度继续补充专业方向和排除清单。

## 五、还必须纳入但容易漏掉的点

1. 院校专业组不是学校整体分数。投档线只代表该专业组的最低投档位置，不代表组内每个专业都能进。
2. 专业组内调剂是最大风险点。只要选择服从调剂，就要默认能接受该组内所有可能被调剂到的专业。
3. 2026 招生计划必须单独核验。历史分数线只能参考，不能替代 2026 年湖北招生计划、专业组代码、计划人数、选科要求和备注。
4. 专业组代码和专业组成可能变化。2023-2025 同名学校的同号/不同号专业组不能机械合并。
5. 选科要求要逐项确认。考生有化学和地理，没有生物；“化学或生物”通常可报，“化学和生物”需要特别警惕并以 2026 计划为准。
6. 体检和限报要提前排查。医学、药学、化工、食品、生物、地理信息、师范、公安、军校、航海、空乘等方向可能涉及色觉、视力、身高、肝功能或其他限制。
7. 公办、民办、独立学院、职业本科、中外合作、预科、民族班、地方专项、国家专项要分开看。名称相近不代表成本、培养质量和录取逻辑相同。
8. 学费和校区会影响真实选择。中外合作、民办本科、分校区、联合培养、产业学院等都需要确认费用和就读地点。
9. 招生计划人数太少的组波动会很大。小计划专业组不宜作为唯一保底。
10. 数学 88、物理 50 提示需要谨慎选择强数学/强物理/强硬件类专业，除非考生本人兴趣和承受力明确。
11. 英语 122 是优势，可以关注语言、国际经贸、跨境电商、师范英语、涉外类方向，但也要看学校平台和就业出口。
12. 医学类、师范类、法学类、会计类、计算机类都要看“后续门槛”。有的专业本科就业一般，需要考研、考编、考证或实习资源支撑。
13. 北京城市偏好要特别冷静。该位次附近北京公办普通本科空间有限，可能伴随高费用、民办、冷门专业、预科或特殊备注。
14. 平行志愿顺序仍然重要。按“最想去且风险可接受”到“稳定保底”排序，不能把不愿意去的组放在前面。
15. 录取联系方式、系统密码、志愿保存、最终确认同样是风险点。志愿填报不是只做方案，还要保证提交动作不出错。

## 六、决策框架

每个候选院校专业组必须形成一条记录：

| 字段 | 要求 |
| --- | --- |
| 年份数据 | 2023、2024、2025 投档线和位次，缺失原因要说明 |
| 2026 计划 | 专业组代码、专业、人数、选科、备注 |
| 学校属性 | 公办/民办/独立/职业本科/中外合作 |
| 城市校区 | 城市、具体校区、交通与生活成本 |
| 专业可接受性 | 可接受专业、勉强接受专业、不能接受专业 |
| 调剂结论 | 是否服从调剂；如果不服从，退档风险如何处理 |
| 风险桶 | 冲、稳、保 |
| 选择理由 | 为什么值得放入志愿表 |
| 排除理由 | 若排除，明确记录为什么 |
| 复核状态 | 未核、CSV 核、原件核、2026 计划核、章程核、最终核 |

## 八、下一步需要家庭补充的信息

这些信息不影响当前数据整理，但会明显影响最终方案：

- 是否只考虑公办本科；民办、独立学院、职业本科、中外合作分别能否接受。
- 家庭年学费和生活费预算上限。
- 是否接受护理、康复、药学、食品、化工、材料、土木、环境、农学、生物类专业。
- 是否偏向师范、医学技术、财经管理、计算机/数据、地理信息、语言类、法学、公务员友好专业。
- 是否接受省内非武汉城市，或重庆、长沙、合肥、南昌、郑州、南京周边等城市。
- 是否有色盲、色弱、近视高度、身高、慢性病等需要避开的体检限制。
- 考生本人最不能接受的专业和城市。
- 对数字媒体技术、师范类、计算机类的真实兴趣排序，以及是否接受相近专业。
- 是否愿意为了城市接受弱一些的学校/专业，或为了公办与专业质量接受非偏好城市。

## 九、保真原则

本项目采用四层保真：

1. 固定事实写入 `candidate-baseline.json`。
2. 官方网页、图片、PDF 原件保存在 `data/official/`。
3. OCR/CSV/筛选结果保存在 `data/derived/`，并标注为派生数据。
4. 重要文件写入 `CHECKSUMS.sha256`，运行 `python3 scripts/verify_baseline.py` 复核。

最终志愿表只在所有候选项完成“官方原件 + 2026 计划 + 招生章程 + 家庭偏好”四项核验后定稿。
