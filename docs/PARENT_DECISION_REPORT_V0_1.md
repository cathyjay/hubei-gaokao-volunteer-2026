# 家长阶段性决策报告 V0.2

生成日期：2026-06-28

报告状态：基于当前稳定数据底座的阶段性决策共识版，不是最终志愿填报表。

适用对象：家庭内部讨论，用于先统一原则、风险边界和下一步核验重点。

隐私说明：本报告只使用分数、位次、选科和家庭偏好等决策信息，不保存姓名、身份证号、准考证号、报名号、截图原件或其他直接身份标识。

## 一、先给家长看的结论

当前最重要的结论不是“马上定 45 个志愿”，而是先让家里对底线和取舍达成一致：

1. 主线应该是公办本科、普通低费用、稳妥录取。
2. 不建议把大量志愿压在高冲学校上，更不能把征集补录当作常规兜底。
3. 学校名气可以适当让一点，换取公办、低费用、专业相关和录取稳定。
4. 专业方向优先看数字媒体技术、计算机科学与技术、软件工程、人工智能、网络/信息安全、电子信息、自动化/机器人工程等相关方向。
5. 家庭中有计算机科学与技术专业背景，可以帮助解释课程、学习强度和后续路径，所以计算机相关方向有额外适配优势，但不是强迫必须选择。

一句话策略：

```text
先保公办低费和不滑档，再争取相关专业和更好学校；少量有证据支撑的冲刺，重点提高稳妥和保底质量。
```

## 四、家庭底线和排序原则

当前家庭底线已经比较清楚：

- 稳第一，不倾向高冲。
- 公办本科优先，普通低费用优先。
- 不希望因为滑档、掉档导致后续补填质量下降。
- 学校可以略降，只要换来更稳的录取、更可接受的专业和更可承受的费用。
- 计算机相关专业辛苦可以接受，但要继续核课程、培养质量、就业/升学路径和完整调剂范围。

建议最终排序优先级：

```text
1. 不滑档、不退档
2. 专业组内所有可能录取或调剂专业可接受
3. 公办、普通低费用、培养模式清楚
4. 专业方向相关，最好有家庭可支持的学习路径
5. 学校平台和城市体验
```

这意味着：如果一个学校看起来更好，但专业组里混有不能接受的专业，或者必须服从调剂而调剂风险说不清，就不应进入最终表。

## 五、对机构报告的评价

机构给河南考生的报告有参考价值，但只适合参考结构，不适合直接照搬结论。

它做得比较好的地方：

- 已经细到院校专业组和组内专业，不是只列学校名。
- 有 2026 招生计划、专业计划数、学费、校区、近年录取数据。
- 会标出概率、冲稳保、院校标签和专业标签。
- 页面和 PDF 的信息展示比较完整，适合家长快速浏览。

它的主要不足：

- “概率”是第三方模型结果，不等于官方录取保证。
- 对新增组、断档、征集、计划变化等风险解释不足。
- 写了“服从调剂”，但没有充分说明组内所有可能调剂专业是否都能接受。
- 对费用、校区、合作办学、企业共建、单科/体检/语种限制的决策影响讲得不够。
- 没有把家庭心理承受、后续费用、是否愿意高冲这些真实约束放到核心位置。

我们的报告应该保留它的信息颗粒度，但要比它多三件事：

1. 每个候选都说明为什么进、为什么暂缓、为什么排除。
2. 每个专业组都看完整调剂范围，而不是只看想填的 6 个专业。
3. 所有最终项都标明证据状态，不能把第三方线索当官方事实。

## 七、当前应当保留的方向

### 1. 普通公办低费用主线

这是主线，也是优先投入核验精力的方向。

保留条件：

- 公办本科。
- 普通学费或低费用。
- 至少命中一个相关专业方向。
- 组内没有明显不能接受专业。
- 历史线处于可冲、可稳或可保范围。
- 校区、学费、选科、体检、语种、单科限制能核清楚。

重点看：

- 计算机科学与技术、软件工程、人工智能、网络工程、信息安全、数据科学与大数据技术。
- 数字媒体技术、电子信息、通信工程、自动化、机器人工程、智能制造。
- 师范类可继续了解，但要看是否真愿意从教、考编压力和就业地区。

### 2. 稳妥和保底质量提升

按最新家庭口径，保底不是“随便有本科就行”，而是要同时满足：

- 公办属性尽量可靠。
- 学费可承受。
- 专业组内调剂也能接受。
- 历史线有足够余量。
- 不依赖征集补录。

当前草表里已有 5 个保底组，但从“稳第一”的家庭偏好看，后续应继续提高保底质量，不能只用数量安慰自己。

### 3. 高费或中外合作专项

高费或中外合作不作为普通主线，只作为专项讨论。

只有满足以下条件才值得继续看：

- 学校或专业明显比普通低费用主线更有优势。
- 总费用、证书、培养模式、校区、出国要求讲得清楚。
- 同组调剂风险能接受。
- 不影响普通低费用主线的稳妥性。

目前更适合把它放在附录，不放进家庭第一决策层。

## 八、当前应当谨慎或暂缓的方向

### 1. 高冲和新增组

不是完全不能冲，但数量要少，且必须有证据支撑。

谨慎原因：

- 新增组缺少同代码历史线，难判断真实热度。
- 高冲学校即使投档线接近，也可能专业不理想。
- 家庭已经明确不希望因为高冲导致滑档或心理压力。

处理建议：

```text
高冲只保留少量“专业组干净、费用正常、就算调剂也能接受”的项。
凡是只是学校名气好，但调剂风险或历史线不清楚的，降级为观察或删除。
```

### 2. 民办、独立学院、职业本科

当前不作为常规主线。

只有在公办保底不足、或者某个项目确实有明显专业/费用/就业优势时，才单独拿出来讨论。不能把它们混在普通公办主线里。

### 3. 医学、护理、动物医学/兽医

当前处理口径：

- 临床医学、口腔医学、中医等医学主线暂缓。
- 护理、助产不进入本轮候选。
- 动物医学、兽医、动物科学不进入本轮候选。
- 医学影像技术、医学检验技术、康复治疗等医技方向只作专项了解，不自动进入主线。

### 4. 组内专业复杂或特殊限制多的专业组

有些专业组虽然命中计算机、电子信息或师范，但组内同时混有色觉限制、体检限制、冷门方向、校区变化或明显不能接受专业。这类组必须先核完整组内专业，再决定是否保留。

## 九、下一轮报告要回答的关键问题

下一版应继续从稳定数据底座生成，不应该只是“候选很多”，而要回答这些问题：

1. 哪些院校专业组可以作为稳妥主线？
2. 哪些院校专业组只是冲刺观察，不能放太多？
3. 哪些院校专业组可以做高质量保底？
4. 每个候选组里，如果没有录到最想去的专业，调剂到其他专业是否还能接受？
5. 学费、校区、培养模式和章程限制是否已经核清楚？
6. 家庭是否接受东北、西北、西南、华北等非最初偏好城市？
7. 中外合作或高收费是否只保留少量专项，还是完全暂缓？

## 十、后续固定报告模板应怎么出

以后每次基础数据底座刷新后，家长报告可以固定按下面结构生成：

1. 家长先看的结论：本轮是否可以缩圈，是否需要增加保底，是否有必须排除项。
2. 数据底座状态：逐专业明细数、专业组数、候选池规模、草表规模、最终门禁状态。
3. 家庭底线变化：公办、费用、专业、地区、调剂、专项是否有变化。
4. 当前候选分层：主线、稳妥、保底、冲刺观察、专项、暂缓。
5. 风险解释：只写可核验风险，例如官方计划未闭环、组内调剂未核、学费/校区/章程限制未核。
6. 家庭待决策问题：需要家长明确接受或不接受的点。
7. 下一步核验清单：按专业组列出谁先核、核什么、核完后能否进入下一版。

这个模板的重点是把“为什么这么选”讲清楚，而不是堆满学校名。

1. 把现有 45 个草表按“稳第一”重新分层，减少高冲和历史待补项。

| 问题 | 建议默认口径 |

| 是否接受学校略差一点换公办低费和稳定录取？ | 接受 |

- 核第 19 期原页或纸质原页。
- 核湖北官方系统或省招办计划。
- 核高校招生章程。
- 核完整院校专业组和调剂范围。
- 核 6 个专业顺序、计划数、学费、选科、校区、语种/单科/体检限制。
- 核近三年投档线和组代码变化。

## 十二、阶段性结论

现在应该给家长看的，不是最终 45 个志愿表，而是这份阶段性决策报告。

这份报告的目的有三个：

1. 让家里先确认“稳第一、公办低费、相关专业优先”的总方向。
2. 让家长理解为什么不能只看学校名气或第三方概率。
3. 为下一版候选压缩和最终填报表设定清晰边界。

后续真正的最终报告，应该在每个候选专业组完成官方计划、原页、章程、费用、校区、完整调剂范围和三年历史线复核后再出。
